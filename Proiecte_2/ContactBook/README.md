📒 Agenda de Contacte — C++ Console App
O aplicație de tip agendă telefonică implementată în C++, care rulează în linia de comandă. Permite gestionarea completă a contactelor cu validare a datelor introduse și persistență locală.
✨ Funcționalități principale

➕ Adăugare contacte cu validare pentru nume, telefon și email
📋 Afișare sortată cronologic (cele mai recente primele)
🗑️ Ștergere cu confirmare
💾 Salvare și încărcare automată din fișier (contacte.txt)
🕐 Timestamp pentru fiecare contact adăugat

🛠️ Tehnologii
C++ STL File I/O OOP-style structs
